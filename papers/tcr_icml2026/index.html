Temporal Consistency Regularization for Catastrophic Forgetting Mitigation in Continual Learning

Y. Matsumoto, R. Takahashi, A. Ogawa, K. Shimizu

Proceedings of the 43rd International Conference on Machine Learning (ICML), 2026

ACCEPTED

Abstract

継続学習における破局的忘却を抑制するため、タスク間の中間層表現の時間的一貫性を正則化項として導入する手法Temporal Consistency Regularization（TCR）を提案する。既存手法はパラメータ空間または出力空間での制約に主眼を置くが、本研究では中間表現の分布シフトを直接抑制することで、旧タスク性能の保持と新タスク学習の柔軟性のトレードオフを改善する。Split-CIFAR100、Split-MiniImageNet、CORe50の3ベンチマークにおいて、EWC、iCaRL、DER++と比較し平均忘却率を1.8〜3.2ポイント改善した。メモリバッファを必要としない設定においてもReplayベース手法に匹敵する性能を示した。

1. Introduction

継続学習において新タスクの学習が旧タスクの性能を劣化させる破局的忘却は長年の課題である[1]。正則化ベース手法[2]はパラメータの重要度に基づき更新を制約するが、タスク間の関係が複雑な場合に柔軟性を失う傾向がある。Replayベース手法[3]は過去データの一部を保持し再学習に用いるが、メモリ制約下では性能が大きく低下する。

本研究の貢献は以下の3点である。(1) 中間層表現の時間的一貫性を利用した新たな正則化手法TCRを提案する。(2) 3つの継続学習ベンチマークにおいて既存手法との系統的比較を行う。(3) メモリバッファなし設定における有効性を検証する。

2. Method

2.1 Temporal Consistency Loss

タスクtの学習時に、タスクt-1終了時点のモデルによる中間層表現と現在のモデルの中間層表現との距離を損失項として加える: ℒTCR = Σl λl·||Φl(t)(x) − Φl(t-1)(x)||²

2.2 Adaptive Weighting

重みλlを層ごとの表現変化量に応じて動的に調整: λl = λ0·(1+γ·ΔΦl)·exp(−α·t)

2.3 Memory Cost

旧モデルのスナップショットのみ保持。Replayバッファと比較して大幅に軽量。

4. Limitations

(1) タスク数が50以上の長期的安定性は未検証。(2) 一貫性損失計算に旧モデルのフォワードパスが必要であり学習時間が約1.6倍に増加。(3) タスク境界が明示的でないオンライン継続学習設定への拡張は今後の課題。

5. Conclusion

中間層表現の時間的一貫性を利用した正則化手法TCRが、メモリバッファなしでも既存のReplayベース手法に匹敵する忘却抑制性能を達成することを示した。学習時間の増加という制約はあるものの、メモリ制約の厳しい実運用環境における適用可能性が高い。
